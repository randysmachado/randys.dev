Quero compartilhar uma dica muito interessante que descobri vasculhando pela internet que é a possibilidade de configurar o VS Code para trabalhar com espaços de trabalho diferentes. 

Tá, mas afinal o que é isso? 🤔

O que é um Workspaces?

Traduzindo a grosso modo é um "espaço de trabalho" do VS Code que é uma coleção de uma ou mais pastas que serão abertas na janela do editor, ou seja, quando tu abre a pasta de um projeto esta pasta se torna teu espaço de trabalho, mas dependendo do fluxo de trabalho algumas configurações atuais do VS Code pode 

pode ser necessário incluir mais de uma pasta neste espaço de trabalho, essa adição pode ser então será necessário utilizar o que eles chamam de **"Espaços de trabalho multi-root"**.
